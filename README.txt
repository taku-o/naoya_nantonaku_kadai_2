どうやる?

        * かんたんなプログラムなので1時間もあれば作れると思います 

    * どういうところを見たいか
        * 可読性
        * オブジェクト指向的なAPIの設計、データと出力処理の分離
        * 可読性、設計的なところが観点で、実装時間や実行スピードは見ません
        * 説明のコメントとかも特に要らない、コードだけあれば ok です
